# VSCode Extension Pack for SDS

This extension pack supports students in the University of
Virginia School of Data Science.

The following extensions are installed in this pack:

- Docker
- GitHub Codespaces
- GitHub Copilot
- Python
- Python Debugger
- Rainbow CSV
- Remote Development
- Remote Containers
- Remote SSH
- Remote WSL
- Remote Explorer
- Remote Server
- VS Live Share
- YAML
